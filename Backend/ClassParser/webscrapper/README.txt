To compile the code
javac -cp jsoup-1.11.3.jar: Scrapper.java

To run the code
java -cp jsoup-1.11.3.jar: Scrapper

This is a webscrapper that will take the html code of a webpage and parse through it.  Jsoup was used for parsing and dynamically updating the page.
/webscrapper/Scrapper.java
A text file was created for each class and semester that contains class information such as department, class number, section, and professor.
